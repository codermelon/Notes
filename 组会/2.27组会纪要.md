# 朱美庆

分享论文：

- A review of deep learning techniques for speech processing

  - 全面概述了关键的深度学习模型及其在语音处理任务中的应用。文章对这些方法进行了分类，并比较了它们在解决语音处理任务方面的优缺点。

  - 广泛涵盖了文献中使用的各种语音处理任务、数据集和基准，并描述了如何利用不同的深度学习网络来处理这些任务。

  - 讨论了深度学习在语音处理中的挑战和未来发展方向，包括对更有效的参数、可解释的模型的需求，以及深度学习在多模态语音处理中的潜力。

- Cross-Lingual Alzheimer’s Disease Detection Based on Paralinguistic and Pre-Trained Features

  - 背景介绍：AD患者的言语具有语言学特征，包括较少的口头真实的词，较多的想象词、停顿和犹豫。
  - 主要贡献：使用副语言和预训练特征进行跨语言阿尔茨海默病检测难点：训练集是英语的，测试集是希腊语目标：ICASSPSPGC-2023 ADReSS-M挑战任务旨在研究如何从语音中提取通用声学特征以进行多语言AD检测。

- Soft-Weighted CrossEntropy Loss for Continuous Alzheimer’s Disease Detection

  - 提出了基于预训练框架 Wav2vec 2.0 和下游模型的阿尔茨海默病语音检测网络，用于改进阿尔茨海默病的检测准确率，实现了 85.45% 的识别准确率。采用软加权交叉熵损失函数替换传统损失函数，优化模型训练过程，提高了模型的泛化能力，并减小了验证集与测试集之间的准确率差异。

- emotion2vec: Self-Supervised Pre-Training for Speech Emotion Representation

  - 提出了emotion2vec，一个通用的语音情感表示模型。（emotion2vec通过自我监督在线蒸馏对262小时未标记的情感数据进行预训练，从而获得通用的情感表示能力。）
  - 证明了在情感预训练中结合话语级损失和帧级损失的策略是有效的。
  - 证明了emotion2vec在语音情感识别任务方面优于最先进的预训练通用模型和情感专家模型。

- Automatic speech-based assessment to discriminate Parkinson’s disease from essential tremor with a cross-language approach

  - 利用高斯混合模型对帕金森病（PD）和特发性震颤（ET）患者进行分类。
  - 研究评估了在德语和西班牙语训练的模型在对捷克语的PD和ET患者进行分类时的领域适应性（跨语言）
  - 证明了韵律和清晰度是PD和ET患者之间鉴别诊断的两个最佳生物标志物

- Vision Mamba: Efficient Visual Representation Learning with BidirectionalState Space Model
  - Mamba面临两个挑战：单向建模和缺乏位置意识。为了应对这些挑战，文章提出了Vision Mamba（Vim）模型，它结合了双向SSM进行数据依赖的全球视觉上下文建模和位置嵌入进行位置感知的视觉识别。
  - 与Transformers类似，Vim可以在大规模无监督的视觉数据上进行预训练，以获得更好的视觉表示。由于Mamba更高的效率，可以进行大规模的预训练。

# 徐鹏义

分享论文：

- 专题一 著名的tranformer模型：
  - PVTv2基于PVTv1的三个缺点加以改进：	
    - 1.计算量大	
    - 2.加强patch之间的联系	
    - 3.取消位置编码，采用DW卷积实现zero-padding位置编码
  - Swin transformer：
    - Swin transformer通过将特征图划分为一个个窗口，将MSA替换为成对的W-MSA和SW-MSA,在减小计算量的同时，也实现了窗口间的一个信息交互。
  - 同时，基于这些transformer模型，也尝试着将他们与U-net网络进行结合，实现医学图像分割任务，如Trans-Unet,swin-Unet等
- 专题二 其他注意力机制：
  - SE-Net提出了一种新的全局注意力模块，即通道注意力模块。
  - CBAM提出了一种通道和空间混合的注意力模块
  - SRM提出了一种基于风格的通道注意力模块，本质上和SE-Net一样，同时采用avgpool和stdpoolECA-Net改进了SE-net中的激活模块，采用了一种可以自适应学习k值的快速1D卷积，实现局部跨通道交互的覆盖。
  - Fca-Net提出了一种新的多谱注意力模块，将X划分为多个部分，选择每个部分对应的2DDCT频率分量相乘，在拼接得到通道特征，来充分获取每个通道多样性的信息。
- 专题三 注意力机制的一些改进方式
  - 目前的模型都采用固定大小的窗口设计，如Swin,限制了他们建模长期依赖关系和适应不同大小对象的能力。VSA提出了一种可变尺寸窗口注意力。
  - CVPT提出了一种隐式条件位置编码CPE，放宽了显式位置编码给输入尺寸带来的限制
  - Agent attention引入了代理向量A，同时作为query和key，实现了softmax attention和linear attention的自然融合。
- 专题四 注意力机制是否真的必要？
  - ConvNeXt以resnet50为baseline,参照swin transformer的架构，采用一切方法向其对其，实现一个完全使用卷积的网络模型，最终结果发现比swin transformer更加优秀
  - Skip attention通过可视化，发现在ViT模型中层与层之间的self attention具有高度相关性，从而导致了冗余计算。通过重用前一层的自注意力计算来近似一个或多个后续层的注意力，为了确保跨层重用self attention不会降低性能，引入了一个简单的参数函数
  - Enjoy-Hamburger：在编码长距离依赖关系和成本方面，self attention并不比矩阵分解（MD）要好，所以采用矩阵分解而不是self attention的方式得到一个全局相关性模块
  - SegNeXt使用廉价卷积运算实现了一种新颖的卷积注意力网络，且效果优于ViT
